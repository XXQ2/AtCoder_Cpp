> # 桁指定について
## **桁指定の方法**
### 1. setprecision
~~~cpp
/**
 * AtCoder ABC159 C問題
 * 正の整数 L が与えられます。 縦、横、高さの長さ (それぞれ、整数でなくてもかまいません) の合計が L の直方体としてありうる体積の最大値を求めてください。
 * **/
int main(){
    double L;
    cin >> L;
    cout << setprecision(10) << (L/3.0)*(L/3.0)*(L/3.0) << endl;
    // または
    printf("%.10\n", (L/3.0)*(L/3.0)*(L/3.0));
    // ちなみに↓だとWAになる
    // cout << (L/3.0)*(L/3.0)*(L/3.0) << endl;
}
 ~~~
